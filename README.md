# FormFlow

FormFlow เป็นระบบแบบฟอร์มออนไลน์ที่พัฒนาขึ้นเพื่อช่วยลดขั้นตอนการรับข้อมูลจากลูกค้า โดยเปลี่ยนจากการให้ลูกค้าส่งข้อมูลผ่านแชท เช่น LINE หรือช่องทางอื่น ๆ มาเป็นการกรอกข้อมูลผ่านเว็บไซต์โดยตรง

แนวคิดของระบบคือให้ลูกค้ากรอกข้อมูลด้วยตนเองตามแบบฟอร์มที่กำหนด ข้อมูลจะถูกบันทึกลงฐานข้อมูลทันที ช่วยลดการคัดลอกข้อมูล ลดข้อผิดพลาดจากการพิมพ์ซ้ำ และทำให้การจัดเก็บข้อมูลเป็นระเบียบมากขึ้น

## วัตถุประสงค์

- ลดการรับข้อมูลผ่านการสนทนาในแอปแชท
- ลดการคัดลอกข้อมูลจากข้อความมาลงเอกสาร
- ลดความผิดพลาดจากการกรอกข้อมูลซ้ำ
- จัดเก็บข้อมูลอย่างเป็นระบบ
- สามารถค้นหาและตรวจสอบข้อมูลย้อนหลังได้
- รองรับการส่งออกข้อมูลเป็นเอกสาร PDF

## การทำงานของระบบ

### ผู้ใช้งาน

- เปิดแบบฟอร์มผ่านเว็บไซต์
- กรอกข้อมูลตามที่กำหนด
- ตรวจสอบข้อมูลก่อนส่ง
- ส่งข้อมูลเข้าสู่ระบบ

ไม่จำเป็นต้องสมัครสมาชิกหรือเข้าสู่ระบบ

### ผู้ดูแลระบบ

- เข้าสู่ระบบ
- ดูรายการข้อมูลทั้งหมด
- ค้นหาข้อมูล
- ดูรายละเอียดแต่ละรายการ
- ส่งออกข้อมูลเป็น PDF
- ดาวน์โหลดเอกสาร

## ตัวอย่างการทำงาน

```
ลูกค้า

เปิดเว็บไซต์
      │
      ▼
เลือกแบบฟอร์ม
      │
      ▼
กรอกข้อมูล
      │
      ▼
กดส่ง
      │
      ▼
บันทึกลงฐานข้อมูล

────────────────────────

ผู้ดูแลระบบ

เข้าสู่ระบบ
      │
      ▼
ดูรายการข้อมูล
      │
      ├── ค้นหา
      ├── ดูรายละเอียด
      └── Export PDF
```

## ประโยชน์ของระบบ

- ลูกค้าสามารถกรอกข้อมูลได้ด้วยตนเอง
- ลดเวลาในการรับข้อมูล
- ลดการพิมพ์ข้อมูลซ้ำ
- ลดข้อผิดพลาดจากการคัดลอกข้อมูล
- จัดเก็บข้อมูลในฐานข้อมูลอย่างเป็นระบบ
- สามารถค้นหาและเรียกดูข้อมูลย้อนหลังได้
- รองรับการนำข้อมูลไปสร้างเอกสาร PDF
